Windows VS Mac - The On Going War 2023

Macbooks

Pros of Macbooks

  1. Powerfull which is amazing for develpers
  2. Light and sleek design
  3. Wide screen display
  4. Amazing battery life
  5. Useful if you have other Apple devices, can all seamlessly sync together

Cons of Macbooks

  1. Expensive ranges from $1000 to $3000 or more depending on the specs
  2. No built in HDMI port
  3. Limited software support
  4. Not great for gaming

Windows

Pros of Windows

  1. User friendly and easy to use
  2. Great for gaming
  3. Compatible for all hardware
  4. Pricing can be quite cheap

Cons of Windows

  3. Compatible for all hardware
